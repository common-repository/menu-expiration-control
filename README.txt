== Description ==

The Menu Expiration Control plugin adds start and expiration dates to WordPress menu items, allowing you to control when each menu item is displayed. This feature is useful for temporary promotions, seasonal menus, or any situation where menu items should only be visible for a specific period.

== Installation ==

1. Upload the `menu-expiration-control` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'Appearance' > 'Menus' section in the WordPress admin area.
4. Add start and expiration dates to your menu items.

== Frequently Asked Questions ==

= What is the purpose of this plugin? =

This plugin allows you to set specific start and expiration dates for menu items, controlling their visibility based on your requirements.

= How can I customize the date formats? =

The date format is set to `YYYY-MM-DD`. Customizations can be made in the plugin code.
